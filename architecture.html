System Architecture

Request Data Flow

1
User Selects Filter
FilterBar.vue dropdowns update reactive refs in useFilters()
2
View Watches Change
View component's watch() triggers, calls getCurrentFilters()
3
API Request Sent
api.js sends GET with query params: ?warehouse=Tokyo&month=2025-03
4
Backend Filters
apply_filters() and filter_by_month() reduce in-memory data
5
Response Rendered
Pydantic validates, JSON returned, Vue reactivity updates DOM

Filter System

Time Period
?month=2025-01
Monthly (2025-01 to 2025-12), Quarterly (Q1 to Q4), or All
Warehouse
?warehouse=Tokyo
San Francisco, London, Tokyo, or All
Category
?category=sensors
Circuit Boards, Sensors, Actuators, Controllers, Power Supplies, or All
Order Status
?status=delivered
Delivered, Shipped, Processing, Backordered, or All
